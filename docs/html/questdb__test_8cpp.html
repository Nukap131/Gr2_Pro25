Detailed Description

Eksempel: send en række til QuestDB via C++ client (ILP/HTTP).

Forbinder til QuestDB containeren på 127.0.0.1:9100 og indsætter en test-række i tabellen trades med symbol, side, pris og amount. Bygning: cmake .. && make (se CMakeLists.txt i samme mappe).
